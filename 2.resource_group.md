# 📂 Azure Resource Group Management

Use the commands below to manage the lifecycle of your resource groups via the Azure CLI.

## :one: Create a Resource Group
To create a new Resource Group, you must provide a unique name and a target Azure region.

### **Syntax**
```bash
az group create --name <resource-group-name> --location <region>
```

### **Example**
```bash
ax group create --name rg-maka --location centralindia
```

## :two: View Resource Groups
Displays all groups in your current subscription
```bash
az group list --output table
```
### **View a Specific Resource Group**
```bash
az group show --name rg-maka
```

## :three: Check Resources inside a Group
The command verifies which resources are inside a group.
```bash
az resource list --resource-group rg-maka --output table
```

## :four: Delete a Resource Group
Deleting a resource group will permanently remove all resources contained within it.
```bash
az resource delete --name <resource-group-name>
```
**Fast Deletion (No confirmation & Background process)** <br/>
```bash
az group delete --name rg-maka --yes --no-wait
```
| Option      | Meaning                    |
| ----------- | -------------------------- |
| `--yes`     | Skip confirmation          |
| `--no-wait` | Run deletion in background |
