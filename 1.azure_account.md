## :lock_with_ink_pen: Azure Login

The `az login` command in the Azure CLI authenticates you to your Azure account so you can run Azure commands in the terminal.

```bash
az login
```
Azure CLI:<br/>
  1. Authenticates with the Azure account.
  2. Generates an access token.
  3. Stores it locally on the machine.
  4. Allows CLI commands to interact with Azure.<br>

The token is saved locally here: <br>
  - Windows
    ```bash
    C:\Users\<username>\.azure\
    ```



## 🔍 Verify Login
The `az account show` command is used to check account info after logging in.

```bash
az account show
```



## :clipboard: List all Subscriptions
```bash
az account list -o table
```



## :unlock: Logout
```
az logout
```
